# Calculadora de IMC em Java ⚖️

Uma aplicação simples desenvolvida em Java para calcular o Índice de Massa Corporal (IMC) do usuário. O projeto utiliza caixas de diálogo (`JOptionPane`) para uma interação mais amigável, recebendo os dados do usuário e exibindo a classificação do IMC com base nas faixas de peso tradicionais.

## 🚀 Funcionalidades

- Recebe o peso e a altura do usuário via interface de janelas (pop-up).
- Calcula o IMC automaticamente.
- Exibe no console o valor numérico exato calculado.
- Mostra uma janela de alerta com a classificação do resultado (Abaixo do peso, Normal, Sobrepeso, Obesidade, etc.).

## 🛠️ Tecnologias Utilizadas

- **Java** - **javax.swing.JOptionPane** (Para interface gráfica de entrada/saída)
- **java.util.Scanner** e **java.util.Locale** (Para formatação padrão americano e leitura de console)
